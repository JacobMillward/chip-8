# Chip-8 Emulator

This project is a Chip-8 emulator written in rust. Mostly written via reading [Cowgod's Chip8 Technical Reference](http://devernay.free.fr/hacks/chip8/C8TECH10.HTM).

## Building

With the Rust Stable toolchain installed:

to build
```
$ cargo build
```

or to build and run 
```
$ cargo run
```
